./data/: `data.win` file. Main game data.

./audiogroup1/: `audiogroup1.dat`, first audio group, `bgm` (background music).

./audiogroup2/: `audiogroup2.dat`, second audio group, `sfx` (sound effects).

data.win has three audio groups:
- audiogroup_default -- stored in data.win,
- bgm -- stored in audiogroup1.dat,
- sfx -- stored in audiogroup2.dat.
